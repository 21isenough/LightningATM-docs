# Lntxbot

## A word of caution

{% hint style="alert" %}
The Lightning Wallet is a "Custodial Bitcoin Wallet". This means, you are trusting those who run the software behind this wallet with Satoshis. Be careful and don't keep too many Satoshis on there.
{% endhint %}
